# odin-recipes
## A Simple HTML Website that Displays Fun Recipes from Star Wars

**Live Site: https://maximustang.github.io/odin-recipes/index.html**

This website utilizes the following HTML concepts:
- HTML boilerplate | `!DOCTYPE`, `meta charset`, `<title>`, etc.
- Text formatting | `<h1>`, `<strong>`, `<em>`, etc.
- Lists | `<ul>`, `<ol>`, `<li>`
- Links | `<a>`
- Images | `<img>`
